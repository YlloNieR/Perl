# Perl
### How to Update Perl Version?
**on Windows**
1. check Version via cmd
```
perl -v & perl6 -v
```
2. update command
```
cpan -a
```

**on Linux**
1. check Version via terminal
```
perl -v
perl6 -v
```
2. update command
```
sudo cpan -a
```

### How to execute Perl programs?
```
perl file.pl
perl6 file.p6
```
